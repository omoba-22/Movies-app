# FilmFury Movies App

A lightweight, fast, and responsive movie listing web app built using HTML, CSS, and JavaScript.
The app fetches live movie data from The Movie Database (TMDb) API and displays popular movies with their posters, titles, and ratings.

# Features

- Fetches real-time movie data from TMDb

- Clean, minimal and responsive UI

- Dynamic movie cards with poster, title & rating

- Fast load time — no frameworks needed

- Beginner-friendly and easy to extend

# Tech Stack

HTML5

CSS3

JavaScript (Vanilla)

TMDb API
